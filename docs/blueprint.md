# **App Name**: ZakhmVerse

## Core Features:

- Poem Generation: Generate a poem in response to the user prompt, adhering to constraints of mood, language, style, length, keywords and rhyme.
- Parameter Setting: UI to adjust mood, language, style, length, keywords and rhyme and send those parameters to the LLM.
- Poem Display: Display the generated poem in a readable format.
- Social Sharing: Enable users to easily share the generated poem on social media platforms.
- History Storage: Save previous poems in User phone.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a contemplative and artistic mood.
- Background color: Light lavender (#E6E6FA) to create a soft, dreamlike canvas.
- Accent color: Muted gold (#B8860B) for highlighting key elements.
- Headline font: 'Literata', a transitional serif with a literary feel. Body font: 'Inter', a grotesque sans-serif with a modern look.
- Use minimalistic icons inspired by classic poetry symbols.
- Simple, spacious layout to focus on the generated text.
- Subtle fade-in animations when the poem is displayed.
